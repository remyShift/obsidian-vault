## Découvertes
- Zed a une **Rules Library** native (palette commandes > "agent: open rules library") qui permet de customiser le prompt de génération de commit sans provider externe
- Les règles globales s'appliquent à tout l'assistant Zed (équivalent d'un system prompt persistant)
- Les règles projet se placent dans `.zed/rules/project.md` au root du repo — Zed les ramasse automatiquement (équivalent de CLAUDE.md pour Cursor)
- On peut mentionner des règles ponctuellement via `@` dans le panel agent
