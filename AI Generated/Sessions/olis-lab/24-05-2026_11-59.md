## Découvertes
- `assertMedia` doit être générique `T extends MediaLike` (structural typing) pour couvrir toutes les collections Upload sans importer les types Payload spécifiques
- `assertMedia` throw sur `url: ''` (falsy) intentionnellement — URL vide aussi inutilisable que null
- Factory closure (`createAsserter`) préférée à une classe — pas de problème de binding `this` à la destructuration
- `Resolved<T>` = `Omit<T, 'url'> & { url: string }` — type trop strict pour les mocks de stories (tous les champs Payload requis), d'où `as unknown as TProduct`
- Le mapper et l'asserter sont orthogonaux : l'asserter fait du type narrowing, le mapper fait de la sélection de champs — les deux couches ont des responsabilités distinctes
- Diego : "make the asserter not handle [EAN]" = ne pas ajouter de logique de sélection de champs dans l'asserter, c'est le rôle du mapper
