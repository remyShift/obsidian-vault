# Recap — olis-lab — 17/04/2026

## Avancées
- Exploration complète du flow Google Merchant Center via analyse du codebase
- Identification de tous les points d'entrée pour le sync produit (Notion button, appels API directs)
- Cartographie de l'architecture en couches : Controller → MerchantsService → GoogleMerchantAdapter → googleMerchantUtils
- Documentation de la transformation des données MongoDB → GMC (champs, mapping langue/pays)
- Identification du flow Google Sheets / Meta : GMC → G-Sheet (manuel) → Meta catalog
- Redaction d'un document Notion complet en anglais pour Michele (data flows, triggers, gaps, next steps)

## Problemes / Bugs
- Aucun bug fixé cette session, mais plusieurs gaps identifiés dans l'existant :
  - Aucune authentification sur les endpoints de sync (`/merchants/*` et `/notion/upsert-*`)
  - Aucun cron job actif en production (tous commentés dans `scheduledJobs.js`)
  - Le bouton Notion ne déclenche plus le sync de manière fiable (automation déconnectée)
  - Pas de retry logic sur les appels GMC échoués

## Decouvertes
- Le pattern adapter (`BaseMerchantAdapter`) est déjà en place pour supporter d'autres marchands (Meta, Amazon) sans toucher la couche service
- Le sync BigBlue est optimisé : une seule requête `/ListInventories` pour tout le batch (pas de requête par SKU)
- Le nettoyage des produits obsolètes est géré automatiquement dans `syncAllLiveProducts` (delete des produits GMC absents des Live products DB)
- Disponibilité calculée avec seuil à 1 : `stock > 1 → "in stock"`, `<= 1 → "out of stock"`
- Deux variantes par produit sur GMC : `online:en:GB:{sku}` et `online:fr:FR:{sku}`
- L'intégration Meta ne passe pas directement par le code — elle passe par un G-Sheet exporté manuellement depuis GMC
- Les credentials Google sont dans `my-olis-lab-keys.json` (non commité, service account GCP)

## Decisions prises
- Aucune décision d'implémentation prise cette session (session investigation + documentation)
- Décision de Michele : créer un projet Notion dédié à la gestion globale des syncs produit
