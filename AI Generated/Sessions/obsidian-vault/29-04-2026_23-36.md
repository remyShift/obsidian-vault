## Découvertes
- Le skill Slidev est stocké dans le vault Obsidian lui-même (`Skills/slidev/`) et non dans `~/.claude/plugins` — il faut le lire manuellement via `ctx_read` si non enregistré dans le harness
- `v-bind(color)` dans les `<style scoped>` Vue fonctionne pour passer des props réactives comme valeurs CSS dynamiques — utilisé pour les couleurs de porte dans `Door.vue`
- La syntaxe `color-mix(in srgb, var(--door-color) 60%, transparent)` permet des glows dynamiques sans avoir à définir des couleurs RGBA fixes par porte
- Les slides Slidev avec `layout: none` supprime tout padding — idéal pour les slides plein écran de section
