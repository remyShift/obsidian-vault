## Avancées

### Nouveaux outils installés (rôle de chacun)
- **ripgrep (`rg`)** v15.1 : remplace/complète `grep -r`, cherche dans le **contenu** des fichiers, respecte `.gitignore`, saute `node_modules`/`dist` automatiquement. C'est le binôme de `fd` (fd = fichiers, rg = contenu). Exemples : `rg "adminLabel"`, `rg useCart -t js -l`.
- **fzf** v0.73 : fuzzy finder. Câblé dans fish (voir Fichiers). Keybinds : `Ctrl+R` (historique cherchable en fuzzy), `Ctrl+T` (insère un chemin de fichier avec preview bat), `Alt+C` (saut de dossier, **dormant sur Mac**). Source = `fd`, previews via `bat`/`lsd`.
- **xh** v0.26 : requêtes HTTP lisibles, alternative moderne à `curl` (réponse colorée/formatée auto). Pour tester l'API v2 Oli's Lab. Ex : `xh localhost:3000/api/products`, `xh POST localhost:3000/api/login email=x@x.com password=secret`.
- **tealdeer (`tldr`)** v1.8 : exemples concrets par commande au lieu du `man`. Cache local à peupler via `tldr --update`. Ex : `tldr git rebase`.
- **btop** v1.4.7 : moniteur système temps réel (CPU par cœur, RAM, réseau, disque, process triables/filtrables). Comble le trou "moniteur live" (ncdu ne fait que le disque statique). Thème **Dracula** (bundlé), fond transparent pour hériter de la translucidité Ghostty. Touches : `f` filtre, `k` tue, souris OK, `q` quitte.

### Helpers git fish : gum → fzf
- **`gco`** (switch de branche) et **`gbd`** (suppression de branche) réécrits en **fzf avec preview du log** (`git log --oneline --color=always -20 {}`, preview-window right 55%). Avantage tueur sur `gum filter` : voir les derniers commits d'une branche en la survolant. Pour `gbd`, on voit ce qu'on jette avant de confirmer.
- `git checkout` → `git switch` (verbe moderne dédié).
- **`gadd`** (multi-sélection de fichiers) et **`gcim`** (saisie du message) **gardés en gum** : c'est leur terrain (multi-select + input texte), fzf n'apporte rien.

## Découvertes
- **fzf 0.73** s'installe dans fish via `fzf --fish | source` (méthode moderne, même logique que `mise activate` / `starship init` / `zoxide init`).
- **Alt+C dormant sur Mac/Ghostty** : Option est réservé aux accents (`macos-option-as-alt = false`), donc le binding fzf de saut de dossier ne se déclenche pas. `Ctrl+R` et `Ctrl+T` marchent. Pas grave : zoxide (`cd`) couvre déjà le besoin. Noté en commentaire dans la config.
- **btop bundle ~45 thèmes** dans `/opt/homebrew/share/btop/themes/`, dont `dracula.theme` → pas besoin de télécharger (respecte le principe "rien en curl").
- **tealdeer** affiche "page not found" tant que `tldr --update` n'a pas peuplé le cache.
