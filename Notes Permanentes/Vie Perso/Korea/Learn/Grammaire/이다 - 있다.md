## 이다 

이다 est un verbe qu'on pourrait traduire en français par le verbe **être**. Attaché à un nom ça permets de le décrire.

> 명사[^1] + 이다

### Conjugaison :


| (présent) | (passé) | (futur) |
| --------- | ------- | ------- |
|           |         |         |





Selon si le nom finit par une consomme ou non on va adapter notre terminaison.

[^1]: Nom
