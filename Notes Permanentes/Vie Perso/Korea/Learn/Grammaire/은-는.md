은/는 est une particule de sujet. Elle s'attache à un nom pour indiquer le thème ou le sujet de la phrase.

> 명사[^1] + 은/는

### Conjugaison :

| | Nom | Nom + 은/는 |
| ---- | --- | -------- |
| 받침 O | 선생님 | 선생님은 |
| 받침 X | 저 | 저는 |

**NB :** On utilise 은 quand le nom se termine par une consonne (받침 O), et 는 quand il se termine par une voyelle (받침 X).

> [!info]- Exemple
> 쯔엉은 학생이에요.
>
> Jeong est étudiant.

[^1]: Nom
