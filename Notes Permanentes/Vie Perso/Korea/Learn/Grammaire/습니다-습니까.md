-습니다/-습니까 sont les terminaisons formelles du coréen. On les utilise pour décrire un verbe ou un adjectif dans un contexte soutenu et poli.

> 동사/형용사[^1] + -습니다 (affirmatif) / -습니까? (interrogatif)

| | Base | -습니까? | -습니다 |
| ---- | --- | -------- | -------- |
| 받침 O | 좋다 | 좋습니까? | 좋습니다 |
| 받침 X | 자다 | 잡니까? | 잡니다 |

**NB :** Quand la base verbale se termine par une voyelle (받침 X), on utilise -ㅂ니다 / -ㅂ니까 (la consonne ㅂ s'ajoute directement à la voyelle finale).

> [!question]- 노래가 좋습니까? 노래가 좋습니다.
> La chanson est-elle bonne ? La chanson est bonne.

[^1]: Verbe ou adjectif
