을/를 est la particule d'objet. Elle s'attache au nom qui est le complément d'objet direct du verbe, et se place juste avant ce verbe.

> 명사[^1] + 을/를

| | Nom | Nom + 을/를 |
| ---- | --- | -------- |
| 받침 O | 밥, 가족 | 밥을, 가족을 |
| 받침 X | 노래, 드라마 | 노래를, 드라마를 |

**NB :** On utilise 을 quand le nom se termine par une consonne (받침 O), et 를 quand il se termine par une voyelle (받침 X).
