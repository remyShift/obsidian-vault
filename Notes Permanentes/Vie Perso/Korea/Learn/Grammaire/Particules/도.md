도 est une particule qui indique une similarité ou une addition. Elle signifie **aussi / également**, et remplace 은/는 ou 이/가 dans la phrase.

> 명사[^1] + 도

| | Nom | Nom + 도 |
| ---- | --- | -------- |
| 받침 O | 음식, 동생 | 음식도, 동생도 |
| 받침 X | 저, 드라마 | 저도, 드라마도 |

**NB :** 도 s'attache directement au nom sans distinction de 받침. Pas d'alternance ici.

> [!question]- 가: 한국 노래를 좋아합니다. 한국 음식도 좋아합니다.
> J'aime les chansons coréennes. J'aime aussi la nourriture coréenne.

> [!question]- 나: 저도 한국 음식을 좋아합니다.
> Moi aussi j'aime la nourriture coréenne.

[^1]: Nom
