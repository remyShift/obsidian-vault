이/가 아니다 sert à nier un fait. C'est la négation de [[이다]], qu'on pourrait traduire par **ne pas être**.

> 명사[^1] + 이/가 아니다

| | Nom | Nom + 이/가 아니에요 |
| ---- | --- | -------- |
| 받침 O | 사람 | 사람이 아니에요 |
| 받침 X | 누나 | 누나가 아니에요 |

**NB :** On utilise 이 quand le nom se termine par une consonne (받침 O), et 가 quand il se termine par une voyelle (받침 X).

> [!info]- Exemple
> 저는 한국 사람이 아니에요. 미국 사람이에요.
>
> Je ne suis pas coréen. Je suis américain.

[^1]: Nom
