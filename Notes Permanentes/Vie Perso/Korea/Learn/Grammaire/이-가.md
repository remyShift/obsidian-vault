이/가 est une particule de sujet. Elle s'attache à un nom pour indiquer le sujet de la phrase.

> 명사[^1] + 이/가

| | Nom | Nom + 이/가 |
| ---- | --- | -------- |
| 받침 O | 동생 | 동생이 |
| 받침 X | 누나 | 누나가 |

**NB :** On utilise 이 quand le nom se termine par une consonne (받침 O), et 가 quand il se termine par une voyelle (받침 X).

> [!info]- Exemple
> 동생이 학생이에요.
>
> Mon petit frère est étudiant.

**Différence avec 은/는 :** 이/가 met l'accent sur le sujet lui-même, tandis que 은/는 introduit le thème général de la phrase.

[^1]: Nom
