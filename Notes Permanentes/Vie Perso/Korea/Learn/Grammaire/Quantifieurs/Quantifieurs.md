Les quantifieurs (의존 명사) sont des mots qui s'intercalent entre un nombre et un nom pour indiquer ce qu'on compte. En coréen, on ne peut pas simplement coller un nombre devant un nom — il faut presque toujours un quantifieur.

Structure : **Nom + Nombre (natif) + Quantifieur**

> 사과 두 개 — deux pommes (et pas ~~두 사과~~)

Tous les quantifieurs de cette liste utilisent les **nombres coréens natifs**, avec les formes contractées suivantes :

| Seul | Devant un quantifieur |
| --- | --- |
| 하나 | 한 |
| 둘 | 두 |
| 셋 | 세 |
| 넷 | 네 |
| 스물 | 스무 (uniquement devant 살) |

## Pour demander la quantité

> 몇 + quantifieur + 이에요?

- 몇 개예요? — C'est combien ?
- 몇 명이에요? — Vous êtes combien ?
- 몇 살이에요? — Vous avez quel âge ?
- 몇 잔이에요? — C'est combien de verres ?
