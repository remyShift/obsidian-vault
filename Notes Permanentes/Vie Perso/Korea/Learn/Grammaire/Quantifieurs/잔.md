잔 est le quantifieur pour compter les **boissons servies dans un verre ou une tasse** (café, thé, eau, bière...).

> 수[^1] + 잔

**NB :** On utilise les nombres coréens natifs. 하나 → 한, 둘 → 두, 셋 → 세, 넷 → 네 devant 잔.

| Quantité | Coréen | Traduction |
| --- | --- | --- |
| 1 verre | 한 잔 | un verre |
| 2 verres | 두 잔 | deux verres |
| 3 verres | 세 잔 | trois verres |

> [!question]- 커피 두 잔 주세요.
> Deux cafés, s'il vous plaît.

> [!question]- 물 한 잔 주세요.
> Un verre d'eau, s'il vous plaît.

**Pour demander la quantité :**

> 몇 잔이에요? — C'est combien de verres ?

**NB :** 잔 s'utilise pour ce qui est servi dans un contenant. Pour des bouteilles entières → 병.

[^1]: Nombre coréen natif
