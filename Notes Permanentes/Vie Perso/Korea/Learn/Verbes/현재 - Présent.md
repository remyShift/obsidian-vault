Comment conjuguer un verbe ou un adjectif **au présent**, dans les deux registres du quotidien : **informel poli** en -아요/-어요, et **formel** en -습니다/-ㅂ니다. On part toujours de la racine (le 원형[^1] moins son 다 final).

> 원형 − 다 → racine → + terminaison

---

## -아요 / -어요 (informel poli)

Utilisé avec un verbe ou un adjectif dans une phrase. Plus informel que -습니다. S'utilise aussi bien pour les questions que les réponses.

| Condition | Exemples | Résultat |
| --------- | -------- | -------- |
| Racine en ㅏ ou ㅗ (sans 하다) | 맑다, 많다 | + 아요 → 맑아요, 많아요 |
| Racine en autre voyelle (sans 하다) | 있다, 없다 | + 어요 → 있어요, 없어요 |
| Verbe en 하다 | 좋아하다, 공부하다 | 하 → 해요 → 좋아해요, 공부해요 |

**Cas irréguliers importants :**

- Racine en **ㅏ** (batchim X) : 가- + 아요 = **가요**
- Racine en **ㅂ** : ㅂ disparaît + 워요 → 덥다 = **더워요**, 춥다 = **추워요**
- Racine en **ㅡ** : ㅡ disparaît, on regarde la voyelle d'avant → 나쁘- + 아요 = **나빠요**
- Racine en **ㅣ** : + 어요 → 이어요 → contracté en **여요** → 가르치- = **가르쳐요**
- Racine en **ㄹ** (batchim) : + 아요/어요 directement → 멀다 = **멀어요**
- **-하다** : -하여요 contracté en **-해요**

---

## -습니다 / -ㅂ니다 (formel)

Utilisé dans les situations formelles : présentations, contextes professionnels, discours.

| Mode | Terminaison |
| ---- | ----------- |
| Déclaratif | -습니다 / -ㅂ니다 |
| Interrogatif | -습니까 / -ㅂ니까 |

| Condition | Exemples | Résultat |
| --------- | -------- | -------- |
| Racine terminée par un batchim (consonne) | 먹다, 읽다 | + 습니다 → 먹습니다, 읽습니다 |
| Racine terminée par une voyelle (ou ㄹ) | 가다, 오다, 살다 | + ㅂ니다 → 갑니다, 옵니다, 삽니다 |

**Cas irréguliers importants :**

- Racine en **ㄹ** : le ㄹ **tombe**, puis + ㅂ니다 → 살다 = **삽니다**, 멀다 = **멉니다**
- Racine en **ㅂ** : ㅂ reste → 춥다 = **춥습니다**, 어렵다 = **어렵습니다**
- **-하다** : → **합니다**

[^1]: forme du dictionnaire (la forme en -다, à conjuguer avant emploi)
