Comment conjuguer un verbe ou un adjectif **au passé**. Le passé se forme avec le marqueur **았 / 었 / 였**, inséré entre la racine et la terminaison. La sélection 았/었 suit **exactement** la même logique que le présent informel (voir [[현재 - Présent]]).

Le réflexe le plus simple : pars de ta forme au présent informel, enlève le 요, ajoute **ㅆ어요**.

> 가요 − 요 + ㅆ어요 → **갔어요**

---

## -았어요 / -었어요 (informel poli)

Même sélection de voyelle qu'au présent : racine en ㅏ/ㅗ → 았, sinon → 었, et 하다 → 했.

| Condition | Exemples | Résultat |
| --------- | -------- | -------- |
| Racine en ㅏ ou ㅗ (sans 하다) | 가다, 좋다 | + 았어요 → 갔어요, 좋았어요 |
| Racine en autre voyelle (sans 하다) | 먹다, 있다 | + 었어요 → 먹었어요, 있었어요 |
| Verbe en 하다 | 공부하다, 좋아하다 | 하 → 했어요 → 공부했어요, 좋아했어요 |

**Cas irréguliers importants :** (mêmes contractions qu'au présent, on ajoute juste ㅆ)

- Racine en **ㅏ** (batchim X) : 가- + 았어요 → contracté = **갔어요**
- Racine en **ㅂ** : ㅂ disparaît + 웠어요 → 덥다 = **더웠어요**, 춥다 = **추웠어요**
- Racine en **ㅡ** : ㅡ disparaît, on regarde la voyelle d'avant → 나쁘- + 았어요 = **나빴어요**, 크- = **컸어요**
- Racine en **ㅣ** : + 었어요 → 이었어요 → contracté en **였어요** → 마시- = **마셨어요**, 가르치- = **가르쳤어요**
- Racine en **ㄹ** (batchim) : + 았/었어요 directement → 멀다 = **멀었어요**, 살다 = **살았어요**
- **-하다** : -하였어요 contracté en **-했어요**

## -았습니다 / -었습니다 (formel)

On prend le radical passé (racine + 았/었/였) et on ajoute 습니다.

| Mode | Terminaison |
| ---- | ----------- |
| Déclaratif | -았습니다 / -었습니다 |
| Interrogatif | -았습니까 / -었습니까 |

| Condition | Exemples | Résultat |
| --------- | -------- | -------- |
| Racine en ㅏ ou ㅗ | 가다, 좋다 | + 았습니다 → 갔습니다, 좋았습니다 |
| Racine en autre voyelle | 먹다, 있다 | + 었습니다 → 먹었습니다, 있었습니다 |
| Verbe en 하다 | 공부하다 | → 했습니다 → 공부했습니다 |

**Le point clé :** au passé formel, la terminaison est **toujours 습니다**, jamais ㅂ니다. Le marqueur du passé (ㅆ) est un batchim, donc la règle "batchim → 습니다" s'applique systématiquement. Conséquence : les verbes irréguliers au présent formel **redeviennent réguliers** au passé.

- 살다 : présent **삽니다** (le ㄹ tombe), passé **살았습니다** (régulier)
- 멀다 : présent **멉니다**, passé **멀었습니다**
- 덥다 : présent 덥습니다, passé **더웠습니다** (l'irrégularité en ㅂ se voit dans le radical 더웠)
