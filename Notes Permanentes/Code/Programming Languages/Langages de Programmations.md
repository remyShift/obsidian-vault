> #LangagesDeProgs 

- Les langages de programmation sont classifier selon leur [[Paradigmes de Programmation]] en fonction de leur structure et l'organisation de ces différents langages pour répondre à un problème. Les [[Structures d'Exécution]] déterminent elles comment les instructions d'un programme sont exécutées par l'ordinateur. 
- Les [[Paradigmes de Programmation]] vont influer sur les [[Structures d'Exécution]].
- En résumé, les [[Paradigmes de Programmation]] fournissent un cadre conceptuel pour organiser le code, tandis que les [[Structures d'Exécution]] déterminent comment ce code est réellement exécuté par l'ordinateur. Ces deux concepts sont étroitement liés et influencent la manière dont les programmes sont conçus, écrits et exécutés.
