**Ruby** est un langage de programmation open-source, simple et intuitif. Il est apprécié pour sa **simplicité** et son **élégance**, tout en restant puissant.

### Caractéristiques principales :

- **Syntaxe claire et lisible** : Ruby est conçu pour être facile à lire et écrire, avec une syntaxe proche du langage naturel.
- **Orienté objet** : Tout est un objet en Ruby, même les nombres et les chaînes de caractères. Cela permet de manipuler des données de manière flexible.
- **Dynamique et interprété** : Ruby ne nécessite pas de compilation. Vous pouvez écrire et exécuter du code directement.
- **Fortement modifiable** : Ruby permet de modifier des classes et objets à la volée (monkey patching), ce qui le rend extrêmement flexible.
- **Large écosystème** : Ruby dispose de nombreux **gems** (bibliothèques) qui facilitent le développement rapide.
