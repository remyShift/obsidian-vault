## sub(replace, by)

Remplace la première occurence d'un motif ou d'un caractère par autre chose.

```ruby
"hello".sub("l", "r") # => "herlo"
```

## gsub(replace, by)

Remplace toutes les occurrences d'un motif ou d'un caractère par autre chose.

```ruby
"hello".gsub("l", "r") # => "herro"
```
