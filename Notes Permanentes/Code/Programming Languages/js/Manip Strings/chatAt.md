## chartAt(index)
Retourne le caractère à l'index spécifié.

```js
let str = "Hello";
let char = str.charAt(1); // "e"
```

## charCodeAt(index)
Retourne le code Unicode du caractère à l'index spécifié.

```js
let str = "Hello";
let code = str.charCodeAt(1); // 101
```
