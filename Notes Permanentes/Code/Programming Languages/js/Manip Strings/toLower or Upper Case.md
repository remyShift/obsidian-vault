## toLowerCase()

Convertit la chaîne en minuscules.

```js
let str = "Hello World";
let lower = str.toLowerCase(); // "hello world"
```

## toUpperCase()

Convertit la chaîne en majuscules.

```js
let str = "Hello World";
let upper = str.toUpperCase(); // "HELLO WORLD"
```
