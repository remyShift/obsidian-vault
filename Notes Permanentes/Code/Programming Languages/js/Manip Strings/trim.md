## trim()

Supprime les espaces blancs des deux côtés de la chaîne.

```js
let str = "  Hello World  ";
let trimmed = str.trim(); // "Hello World"
```

## trimStart()

Supprime les espaces blancs du début de la chaîne.

```js
let str = "  Hello World  ";
let trimmedStart = str.trimStart(); // "Hello World  "
```

## trimEnd()

Supprime les espaces blancs de la fin de la chaîne.

```js
let str = "  Hello World  ";
let trimmedEnd = str.trimEnd(); // "  Hello World"
```
