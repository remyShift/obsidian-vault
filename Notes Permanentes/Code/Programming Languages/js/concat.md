## concat(...strings)

Concatène une ou plusieurs chaînes de caractères à la chaîne d'origine.

```js
let str1 = "Hello";
let str2 = "World";
let result = str1.concat(" ", str2); // "Hello World"
```

Marche aussi sur un tableau :

```js
let arr1 = [1, 2];
let arr2 = [3, 4];
let arr3 = arr1.concat(arr2); // [1, 2, 3, 4]
```
