## slice(beginIndex, endIndex)

Extrait une section de la chaîne et retourne une nouvelle chaîne.

```js
let str = "Hello World";
let sliced = str.slice(0, 5); // "Hello"
```

Marche aussi sur un tableau :

```js
let arr = [1, 2, 3, 4, 5];
let sliced = arr.slice(1, 3); // [2, 3]
```
