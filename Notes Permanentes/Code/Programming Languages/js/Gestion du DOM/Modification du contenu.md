## element.textContent

Obtient ou définit le contenu textuel d'un élément.

```js
const element = document.getElementById('myId');
element.textContent = 'Nouveau texte';
```

## element.innerHTML

Obtient ou définit le contenu HTML d'un élément.

```js
const element = document.getElementById('myId');
element.innerHTML = '<strong>Nouveau contenu HTML</strong>';
```

## element.innerText

Obtient ou définit le texte d'un élément en prenant en compte les styles CSS.

```js
const element = document.getElementById('myId');
element.innerText = 'Nouveau texte stylé';
```
