## pop()

Supprime le dernier élément d'un tableau et retourne cet élément.

```js
let arr = [1, 2, 3];
let last = arr.pop(); // [1, 2], last = 3
```

## shift()

Supprime le premier élément d'un tableau et retourne cet élément.

```js
let arr = [1, 2, 3];
let first = arr.shift(); // [2, 3], first = 1
```

### unshift()

Ajoute un ou plusieurs éléments au début d'un tableau et retourne la nouvelle longueur du tableau.

```js
let arr = [1, 2, 3];
arr.unshift(0); // [0, 1, 2, 3]
```
