Voici une liste des commandes courantes que tu utiliseras avec Expo CLI pendant le développement de ton application :

| Command                         | Description                                                                                                                                                                                                    |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `npx expo start`                | Démarre le serveur de développement (si on utilise Expo Build)                                                                                                                                                 |
| `npx expo prebuild`             | Crée des dossiers et fichiers de [Prebuild](https://docs.expo.dev/workflow/prebuild/).                                                                                                                         |
| `npx expo run:android`          | Compile nativement l'app Android de manière locale.                                                                                                                                                            |
| `npx expo run:ios`              | Compile nativement l l'app iOS app de manière locale.                                                                                                                                                          |
| `npx expo install package-name` | Utilisé pour installer de nouvelle librairie / dépendances dans notre projet.                                                                                                                                  |
| `npx expo lint`                 | [Setup and configures](https://docs.expo.dev/guides/using-eslint/) ESLint. Si ESLint est déjà installé cette commande va [lint les fichiers de mon projets](https://docs.expo.dev/guides/using-eslint/#usage). |
| `npx expo-doctor`               | Pour avoir un diagnostic des problèmes de notre app.                                                                                                                                                           |
