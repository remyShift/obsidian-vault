Le component `<ActivityIndicator/>` permets d'indiquer à l'utilisateur qu'un processus de chargement est en cours (// loader).
- On peut gérer le fait qu'il soit display ou non grâce à la props `animating` qui par défaut est à true, avec un state on peut ensuite gérer la valeur de `animating` est donc display ou non notre composant.

On peut le styliser aussi que ça soit la couleur ou encore sa taille.
