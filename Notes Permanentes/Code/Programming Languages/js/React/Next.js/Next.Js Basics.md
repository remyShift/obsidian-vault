**NextJS =** [[React Basics | React]] Framework pour faire des web apps. Ce qui veut dire qu'il utilise React pour l'UI mais il comprends d'autres features plus facile a mettre à mettre en place qu'en React, où on aurait du installer des packages externes, comme :
- Routing,
- Optimisation du rendu,
- Data fetching,
- Bundling,
- Gestion du SEO,
- ...

Pour se setup cf la [Docs | Next.js](https://nextjs.org/docs).
```
npx create-next-app@latest
```


Contrairement à React qui n'est que de l'UI (// que du front), Next ajoute la possibilité d'avoir des [[React Server Components]].

NextJS inclu la possibilité d'avoir un[[Routing NextJS | routing]] entre les pages fait via notre organisation de fichier.

De plus NextJS permets de gérer les [[Server Actions]] directement grâce aux servers components.
