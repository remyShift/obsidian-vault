NextJS permets de faire des actions RESTful sur le serveur au travers des différentes [[Méthodes HTTP]].

L'avantage en NextJS est que nous n'avons pas besoin de configurer un serveur pour ce faire car il fournit le file [[Route file |route.ts]] qui permets de gérer ça.
