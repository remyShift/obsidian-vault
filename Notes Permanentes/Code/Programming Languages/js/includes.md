## includes(searchString, position)

Détermine si une chaîne contient une sous-chaîne à partir de la position donnée (par défaut 0).

```js
let str = "Hello World";
let contains = str.includes("World"); // true
```

Marche aussi pour un tableau :

```js
let arr = [1, 2, 3];
let hasTwo = arr.includes(2); // true
```
