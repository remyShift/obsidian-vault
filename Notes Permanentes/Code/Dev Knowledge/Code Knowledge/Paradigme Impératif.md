#### Fonctions :
- Signature : éléments qui définissent notre fonction à savoir son nom, ses paramètres et ce qu'elle retourne
	- permets de déclarer ce que notre fonction fait et à d'autres partis de notre code d'appeler cette fonction par son nom et ses paramètres sans forcément connaitre son fonctionnement,
- Déclaration : création de notre fonction et les éléments qui la compose,
- Paramètres : valeurs qui vont être utilisé dans la fonction,
- Arguments : valeurs qui vont être utilisé lors de l'appel de la fonction et passé en paramètres,
- Récursivité : Principe d'appeler une fonction au sein d'elle même,
- Fonction anonyme : fonction qui n'a pas de nom (ex fonction fléchée),
	- Fonction fléchée : syntaxe concise / compact pour définir des fonctions courtes
		- généralement attribuée à des constantes,
```
const sum = (a, b) => {
	return a + b; 
}
```
