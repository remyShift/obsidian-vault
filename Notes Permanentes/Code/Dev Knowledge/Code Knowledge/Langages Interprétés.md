- Langage compilé : JS, Ruby … 
- Compilation faites ligne par ligne du code directement lors de l'exécution,
- **Avantages :**
	- Pas de temps de compilation,
	- Cross-Plateforme (si un logiciel interpréteur est sur la plateforme),
- **Inconvénients :**
	- Exécution + lente,
	- Code doit être partagé pour qu'une tierce personne puisse l'exécuter aussi.
