- Langage compilé : C, C++ ...  
- Compilation : transforme du code compréhensible pour l'Homme par du code compréhensible par un ordi,
	- Une fois la compilation faites on a un fichier exécutable que notre ordi pourra comprendre et exécuter.
- **Avantages :** 
	- Exécution rapide,
	- Reverse Engineering + complexe,
- **Inconvénient :**
	- Pas de Cross-Plateforme,
	- On doit re-compiler a chaque modification.
