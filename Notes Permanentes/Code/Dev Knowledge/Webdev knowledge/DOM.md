- Le DOM, ou **D**ocument **O**bject **M**odel, est essentiellement une représentation hiérarchique des éléments d'une page Web sous forme d'objets manipulables par les programmes informatiques. Cela permet aux développeurs de créer des scripts qui peuvent accéder, modifier et ajouter du contenu dynamiquement sur une page Web, offrant ainsi une expérience interactive aux utilisateurs.
