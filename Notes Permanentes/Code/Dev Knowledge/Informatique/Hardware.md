> #Informatique 

Ensemble physique de composants (ex : ordi, téléphone, ....).

Dans un ordinateur on retrouve différents composants :
##### Les périphériques :
- D'entrés
	- Souris,
	- Manette.
	- Clavier  
- De sortie
	- Écran,
	- Casque

##### Les composants :
- Processeur (CPU) : reçoit des instructions, les exécutent et retourne le résultat,
- Carte Graphique (GPU),
- Mémoire Vive (RAM) : stock les instructions à faire pour le CPU et les réponses de ce dernier,
- Disque dur (HDD ou SSD),
- Carte mère (éléments principal car c'est elle qui va faire la liaisons entre tout les composants)
