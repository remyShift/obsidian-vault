## Les voyelles simples

Le hangeul est composé de 10 voyelles simples qui sont séparées en 2 groupes : les voyelles verticales et les voyelles horizontales. 

Les voyelles ne peuvent s'écrire seule pour former un syllable, elles doivent être précédé d'une consonne.

Savoir distinguer le sens d'une voyelle est essentiel car cela permets de savoir où est placé la consonne qui la précède.

### Les voyelles verticales

Elle se place à droite de la consonne qui la précède.

Exemple : le mot "dent" qui se prononce {i}, il faut donc la consonne muette ㅇ + la voyelle l = 이 

```
ㅏ --> {A}
```

```
ㅑ--> {YA}
```

```
ㅓ --> {O} ("O" ouvert)
```

```
ㅕ --> {YO}
```

```
ㅣ --> {I}
```

### Les voyelles horizontales

Elle se place sous la consonne qui la précède.

Exemple : le chiffre 5, se prononce {O}. Il faut donc la consonne muette ㅇ + ㅗ = 오 

```
ㅗ --> {Ô} 
```

```
ㅛ --> {YÔ}
```

```
ㅜ --> {OU}
```

```
ㅠ --> {YOU}
```

```
ㅡ --> {EU}
```
